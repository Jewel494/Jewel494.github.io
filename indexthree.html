Blog Post: December 6th, 2018

Hi there, thank you for coming to our site. So I'm going start off by saying that you should always treat the cats you see with care and respect. It is very important to do this because this is too often forgotten, either by carelessness or actually meaning to hurt a cat. Treat the cats as you would want to be treated. If you are taking care of a cat, make sure the cat has everything it needs from food to water to attention. If a cat is in need, help it just like you would with a person. If you haven't gotten the point already, just make sure you treat all the cats that you encounter very well.
